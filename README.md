# ProductManagement
A simple spring boot Product Management application with advanced features performing CRUD operations.

1.Integrated with Mysql database and Performed Crud operations using Spring Data JPA.
2.Written test cases for controller methods using Mockito for mocking for the repository.
3.Created Spring boot profiles, dev and prod.
4.Logged method execution time using Spring Aspect.
5.Implemented Global exception handler to handle exceptions thrown by CRUD operations.
6.Create Swagger API documentation for all the endpoints.
